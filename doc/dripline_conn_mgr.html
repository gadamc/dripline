Description

dripline_conn_mgr is one of the dripline core 'managers'. it has a very simple task - hand out connections to the database when it is asked to. for now it is incredibly simple and just provides a connection to anybody, anytime. this may require some form of connection pooling in the future - this module is designed to guarantee an API.
