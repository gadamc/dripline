Description

dripline_conf_mgr is one of the core dripline 'managers' and is probably the most important module in the entire codebase. it stores all of the state regarding the current configuration, and is able to generate executable fun()s from that state. whenever a query or command is sent through the database, calls are made to this module in order to determine what to do.
